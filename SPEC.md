# WHM Mini - Lightweight CI/CD Health Monitor

## Project Overview
- **Project Name**: WHM Mini
- **Type**: CLI Tool (Lightweight)
- **Core Functionality**: A simplified, fast CLI tool that provides quick health checks for CI/CD pipelines without the full analysis overhead
- **Target Users**: Developers who want quick pipeline status checks

## Features
- Quick status check (pass/fail)
- Basic success rate percentage
- Last run duration
- Minimal output - just the essentials
- Lightning fast execution
